Detailed Description

A Json file wrapper class for storing and retrieving key-value pairs in non-volatile memory.

Includes support for:

  • ESP8266 && ESP32 (tested):
    • Makes use of the SPIFFS(ESP8266) and SPIFFS(ESP32) libraries to create, access (read/write) and manage a given Json file in the filesystem.
  • Generic Arduino (untested):
    • Should work with all Arduino platforms, but has not been tested!

Set the value of a given entry.

Set the value of a given entry with key key to value.

If \key matches an existing entry, its value will be overwritted by value. If value="", any matched entry will be removed from memory, i.e. the function becomes equivalent to removeEntry(key). If key is not matched against any entry, a new entry will be created.

Note: A value="" will cause any existing entry to be removed.
